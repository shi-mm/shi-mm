重复的数据分两种，一种是表中部分字段的重复，一种是两行以上的记录完全一样。

\[ Q^\prime(i,j) = q(i,j)\alpha(i,j), Q^\prime(j,i) = q(j,i)\alpha(j,i) \]

部分字段的重复

1 查询不重复的数据SQL：

select 字段1,字段2,count() from 表名 group by 字段1,字段2 having count() = 1
